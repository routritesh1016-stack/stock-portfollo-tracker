# Stock Portfolio Tracker

## Description
This is a simple Python project that calculates the total investment value of a stock portfolio using manually defined stock prices.

The user enters stock names and quantities, and the program calculates the total investment amount.

---

## Features
- User can enter stock names and quantities
- Uses a hardcoded dictionary for stock prices
- Calculates total investment value
- Displays portfolio details
- Saves output to a `.txt` file (optional)

---

## Technologies Used
- Python
- Dictionary
- Input/Output
- Basic Arithmetic
- File Handling
